Lesson 4: Logical Conditions and Conditional Statements

Logical Conditions

Let's dive into logical conditions, which are like mathematical operations to compare values:

  • Equals: a == b
  • Not Equals: a != b
  • Less Than: a < b
  • Less Than or Equal To: a <= b
  • Greater Than: a > b
  • Greater Than or Equal To: a >= b

Conditional Statements: if Statements

In Python, we use logical conditions to make decisions using if statements. These statements help the computer decide what to do based on certain conditions:


            age = 12
            if age < 18:
                print("You are a kid")

else Statement

We can also include an else statement to provide an alternative action:


            age = 12
            if age < 18:
                print("You are a kid")
            else:
                print("You are an adult")

elif Statement

And for more complex decisions, we can use elif (short for "else if") to test multiple conditions:


            age = 12
            if age < 18:
                print("You are a kid")
            elif age == 17:
                print("You are almost becoming an adult")
            else:
                print("You are an adult")

Lesson 5: Loops - for and while

for Loop

The for loop allows us to repeat actions a certain number of times or iterate through a collection of items. Here are some examples:


        friend = "Joe"
        for i in range(10):
            print("Hello,", friend)
        

        friends = ["Alice", "Joe", "Alex"]
        for friend in friends:
            print(friend)

continue

The continue statement allows you to skip the current iteration and continue with the next one:


        friends = ["Alice", "Joe", "Alex"]
        for friend in friends:
            if friend == "Joe":
                continue
            print(friend)

break

The break statement is used to stop the loop prematurely:


        friends = ["Alice", "Joe", "Alex"]
        for friend in friends:
            if friend == "Joe":
                break
            print(friend)

while Loop

The while loop keeps going until a certain condition is no longer true:


        count = 0
        while count < 5:
            print(count)
            count += 1

In the code above, we define a function called average that takes a list of numbers as its argument. Inside the function, we calculate the total sum of the numbers using the sum() function and then divide it by the length of the list to compute the average. Finally, we call the average function with the grades list and store the result in average_grade.

The average grade is then printed to the screen.

Lesson 14: Introduction to Errors

Understanding Errors in Python

Errors are a natural part of programming. When you write code, you may encounter different types of errors. In this lesson, we'll explore two common types of errors: syntax errors and runtime errors.

Syntax Errors

Syntax errors are errors that occur when you don't follow the correct syntax or rules of Python. These errors are typically detected by the Python interpreter before your program runs. They prevent your code from even starting.

Example of a syntax error:


        print("Hello, World")
        x = 10
        for i in range(5)
            print(i)

In the code above, there are two syntax errors. The first one is the missing colon at the end of the "for" statement, and the second one is the missing indentation for the "print" statement inside the loop. These errors need to be fixed before the program can run.

Runtime Errors

Runtime errors, also known as exceptions, occur when your code is syntactically correct, but something goes wrong while the program is running. These errors can include division by zero, trying to access an element that doesn't exist in a list, or using a variable that hasn't been defined.

Example of a runtime error:


        num1 = 10
        num2 = 0
        result = num1 / num2

In the code above, a division by zero occurs, which is a runtime error. This error is detected while the program is running, and it causes the program to terminate with an error message.

Handling Errors

Python provides ways to handle errors gracefully using try and except blocks. These blocks allow you to catch and handle exceptions without causing your program to crash.

Example of error handling:


        try:
            num1 = 10
            num2 = 0
            result = num1 / num2
        except ZeroDivisionError:
            print("Error: Division by zero is not allowed.")

In this example, we use a try-except block to catch the ZeroDivisionError that occurs when dividing by zero. Instead of crashing, the program displays an error message and continues running.

Lesson 15: Debugging Techniques

Debugging: Finding and Fixing Errors

Debugging is the process of finding and fixing errors in your code. It's an essential skill for any programmer because, in the real world, code almost always contains bugs. In this lesson, we'll explore some strategies for debugging and practice fixing common programming errors.

Common Types of Errors

Before we dive into debugging techniques, let's review some common types of errors:

  • Syntax Errors: These errors occur due to violations of Python's syntax rules. They are detected by the interpreter before the program runs.
  • Runtime Errors: Also known as exceptions, these errors occur during program execution when something unexpected happens.
  • Logical Errors: These errors do not generate error messages, but they cause the program to produce incorrect results.

Debugging Strategies

Here are some debugging strategies to help you identify and fix errors:

  1. Read Error Messages: When an error occurs, Python often provides an error message with useful information. Pay attention to these messages as they can point you to the problem.
  2. Review Your Code: Carefully review your code, line by line, to spot any obvious mistakes or typos.
  3. Use Print Statements: Insert print statements at different points in your code to display variable values and control flow. This can help you understand what's happening.
  4. Comment Out Code: Temporarily comment out sections of your code to isolate the problem area. This can help you narrow down the source of the error.
  5. Google and Documentation: Don't hesitate to search online for solutions or check Python's official documentation for guidance.
  6. Ask for Help: If you're stuck and can't find a solution, ask for help from a peer, teacher, or online programming community.

Debugging Exercise

Lesson 16: Strings in Python

What Are Strings?

In Python, a string is a sequence of characters enclosed in either single (' ') or double (" ") quotes. Strings are used to represent text and are an essential part of programming because they allow you to work with words, sentences, and more.

String Manipulation

Strings can be manipulated in various ways. Here are some common string operations:

Concatenation

Concatenation is the process of combining two or more strings to create a new string. You can concatenate strings using the "+" operator:


        first_name = "John"
        last_name = "Doe"
        full_name = first_name + " " + last_name
        print(full_name)

Slicing

Slicing allows you to extract a portion of a string. You specify a starting and ending index, and Python gives you the characters within that range:


        greeting = "Hello, World"
        print(greeting[0:5])  # Output: "Hello"

Length

You can find the length of a string (the number of characters it contains) using the len() function:


        text = "Python is amazing"
        length = len(text)
        print("The length of the string is:", length)

Lesson 17: String Methods

String Methods in Python

Python provides several built-in methods that allow you to manipulate and transform strings. In this lesson, we'll explore some commonly used string methods:

upper()

The upper() method converts all characters in a string to uppercase:


        text = "python programming"
        print(text.upper())  # Output: "PYTHON PROGRAMMING"

lower()

The lower() method converts all characters in a string to lowercase:


        text = "Python Programming"
        print(text.lower())  # Output: "python programming"

split()

The split() method divides a string into a list of substrings based on a specified delimiter. By default, it splits the string at whitespace characters:


        sentence = "Python is fun"
        words = sentence.split()
        print(words)  # Output: ['Python', 'is', 'fun']
